Pasos para ejecutar el programa:
1. Crear un archivo de texto plano .txt en la ruta donde se encuentre el código.
2. Dentro de él colocar de la siguiente manera los siguientes parámetros: 
   <<nombre del archivo a crear>>.wav
   frecuencia de muestreo, canales (en este caso 1)
   frecuencia a escuchar
   tiempo
3. Abrir símbolo de sistema y colocarse en la ruta donde se encuentra el código y
   el archivo .txt.
4. Ingresar el comando <<javac *.java>> para compilar todas las clases.
5. Una vez compilado ingrese el comando <<java EjecutaWAV.java>> para ejecutar la clase
   principal.
6. Si la ejecucuión fue un éxito en la ruta actual se habrá creado un archivo con
   extensión .wav.
7. Para probarlo se puede desde el reproductor que tengas por defecto y si quieres ver
la onda generada puedes utilizar el software Audacity.
